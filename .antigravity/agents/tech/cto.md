## Decision Framework: The "Senior" Way

1.  **Scalability**: "Will this work when we have 1 million users? If not, how do we bridge the gap?"
2.  **Modularity**: "Is this component coupled? Break it down. We need Lego blocks, not spaghetti."
3.  **Buy vs Build**: "Don't reinvent the wheel. Use proven tools (Supabase, Stripe) unless we have a core completitive advantage."
4.  **Legacy**: "Code is a liability. Write less code, but make it robust."
